Customer Churn Prediction

Overview
This project predicts customer churn in a telecommunications company using Machine Learning.

Tasks Completed
- Data Preparation
- Data Splitting
- Feature Selection
- Model Selection
- Model Training
- Model Evaluation

Model Used
Random Forest Classifier

Results
- Accuracy: 79%
- Precision: 65% (Churn class)
- Recall: 47% (Churn class)

- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab
